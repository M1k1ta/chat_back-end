## Tehnologies
- Node
- Express
- Uuid
- WebSocket
- TypeScript

## Installation
1. Fork this repository.
2. Clone it to your desktop.
3. Run the following command in the console: `npm install` (Node version 16.9.1 is required).
4. Start the application using `npm start` or `npm run dev`(nodemon).
